Learn with Error

You can randomly generate a vector \(A\) of random number and a noise vector \(n\) with a secret value \(s\) to create \[ B=As+n \]

expand for full code 
A=[80, 86, 19, 62, 2, 83, 25, 47, 20, 58, 45, 15, 30, 68, 4, 13, 8, 6, 42, 92]
e=[3, 3, 4, 1, 3, 3, 4, 4, 1, 4, 3, 3, 2, 2, 3, 2, 4, 4, 1, 3]
s=5
q = 97
B=[]
for x in range(0,len(A)):
   B.append((A[x]*s+e[x])%q)
print(">> B={}".format(B))
>> B=[15, 45, 2, 20, 13, 30, 32, 45, 4, 3, 34, 78, 55, 51, 23, 67, 44, 34, 17, 75]

Now, \(A\) and \(B\) are made as public keys while \(s\) is the secret private key

Encryption for a message \(M\)

\[\begin{eqnarray*} u&=&\left(\sum_{i}^{} {A_i}\right) \mod q\\ v&=&\left(\sum_{i}^{}{B_i} +\left\lfloor\frac{q}{2}M\right\rfloor \right) \mod q \end{eqnarray*}\]
expand for full code 
import math
u,v=[0,0] 
M=0
for x in range(0,len(A)):
   u=(u+A[x])%q
   v=v+math.floor(q/2)*M

print(">> u:{}, v:{}".format(u,v))
>> u:29, v:0

Decryption

You can create \[ d = (v-su) \mod q \]

\[ \bar{m}=\begin{cases} 1 & d <\frac{q}{2},\\ 0 & d>\frac{q}{2} \end{cases} \]

import math
print(">> u:{}, v:{}, s:{}".format(u,v, s))
d = (v-s*u) %q
mbar = 1 if d < q/2 else 0
print(">> d:{}, m:{}, orginal message m:{}".format(d,mbar,M))
>> u:29, v:0, s:5
>> d:49, m:0, orginal message m:0
